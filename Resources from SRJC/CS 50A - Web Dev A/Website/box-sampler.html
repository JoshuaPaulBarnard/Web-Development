Examples of Box Models:

Content

This is a sample within a box model. The areas within this model are "content", "padding", "border" and "margin". This box only contains text and a green background, without any additional formatting.

Content + padding

This is a sample of text which within a box model with padding. The areas within this model are "content", "padding", "border" and "margin". Padding adds blank space surrounding all 4 sides of the content, and this space displays the background color of the content.

Content + padding + border

This is a sample of text about the box model with content, padding, and a border. The areas within this model are "content", "padding", "border" and "margin". This box adds a white border of 2 pixels.

Content + padding + border + margin

This is a sample of text about the box model with content, padding, a border, and margins. The areas within this model are "content", "padding", "border" and "margin". This box adds an additional 50 pixel margin on all 4 sides of the content which will display in the pages default background color.
